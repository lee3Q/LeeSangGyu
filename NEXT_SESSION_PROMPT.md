## 두 가지 모드 전환: 해 / 달

### 달 (현재 = 미친 버전)
- 다크 배경, 뇌/돌멩이/나뭇잎/별 떠있는 공간
- 클릭해서 탐험
- 기괴하고 중독성 있는 디자인
- 현재 만든 것과 같은 방향

### 해 (바른 버전)
- **깔끔하게 흰 바탕에 검은 글자. 전혀 안 기괴하게.**
- 포트폴리오/취업지원서 느낌
- **스크롤 방식** (기존 포트폴리오처럼)
- 내가 만든 것 + 내 스펙 위주
- "나의 뇌" 정보는 빼고. 깔끔하게 프로젝트와 스킬만.
- 구조:
  - 이름 + 한 줄 소개
  - 프로젝트 (카드 형식, 링크 포함)
  - 스킬/도구
  - 연락처

### 전환 방식
- 화면 어딘가(우상단?)에 **해/달 토글** 아이콘
- 해 아이콘 누르면 → 미친 버전 (달 모드)
- 달 아이콘 누르면 → 바른 버전 (해 모드)
- 전환 시 부드러운 페이드 트랜지션
