# 이상규 월드 v4 — 설계 문서

## 핵심 변경 (v3.1 → v4)

### 1. 돌멩이/나뭇잎 → 사람/동물
- 돌멩이(12개) → **사람 SVG** (디테일한 선화, 호버시 입 벌어짐)
- 나뭇잎(9개) → **동물 SVG** (디테일한 선화, 호버시 입 벌어짐)
- 뇌도 중앙 고정이 아니라 **자유 배회**
- 별(3개) → 유지하되 사람 SVG로 교체 (연락처 담당)

### 2. "가만히 있어" 버튼
- 버튼 누르면 **모든 캐릭터(뇌 포함)에 눈(검은 동공)이 생김**
- 모든 캐릭터 **제자리에서 멈춤** (animation-play-state: paused)
- **동공이 커서를 따라감** (눈동자 추적 효과)
- 루시드 드림 느낌 — 꿈인 걸 인지하니 모두가 나를 쳐다보는 느낌
- 다시 누르면 원래 상태 복귀

### 3. 캐릭터 클릭 내용 확장
- 단순 인용문이 아니라 **왜 그렇게 생각하는지** 설명 포함
- 노무사: "노무사와 인간의 접점" — 노동법을 사람 이해의 렌즈로
- 분석: "고g가 감정을 잘못 진단하는 구조" (자기분석)
- 기술 용어: 일상어로 해설 (커스텀 스킬 = "AI에게 내 방식을 가르친 것" 등)

### 4. 송명기님
- 커피챗 내용 제거
- 그냥 **손 흔드는 사람** 하나 (다른 사람들 사이에 섞여 있음)
- 클릭하면 "송명기 대표님, 반갑습니다!" 정도만

### 5. 해/달 토글
- **이모지 금지** — SVG 코드로 그린 해/달 아이콘
- **기본 모드 = 해 (바른 버전)**

### 6. 달 모드 (진짜 나)
- 사람/동물이 떠다니는 공간
- 클릭해서 탐험
- 기괴하고 몽환적

### 7. 해 모드 (대외용)
- 깔끔한 포트폴리오/이력서 느낌
- 프로젝트 + 스킬 + 연락처
- 스크롤 방식

## 기술 제약
- 단일 index.html (CSS/JS 인라인)
- 외부 의존성: 구글 폰트 CDN만
- 모바일 우선
- Canvas 파티클 배경 유지 (달 모드)
