# 이상규 월드 v5 — 설계 문서

## v4 → v5 핵심 변경

### 1. 캐릭터: 졸라맨 → 둥실 얼굴
- 전신 선화 → **둥근 얼굴만** 떠다님 (몸 없음)
- 헤어 스타일 12종 (short/long/wave/cap/bald/messy/ponytail/afro/flat/horn/halo/bangs/parted)
- 각 캐릭터마다 다른 스킨톤, 헤어 색, 표정
- 강세 장식(glasses/mole/earring/freckles) 매칭
- 동물도 귀 포함 얼굴만 (고양이/강아지/새/올빼미/여우/토끼/곰/물고기/거북)

### 2. 눈 개선
- 큰 흰자 + 또렷한 동공 + **하이라이트 점**
- "가만히 있어" 모드에서 **동공 이동 범위 2배** (2.2px → 4px)
- 하이라이트도 미세하게 따라 움직임
- 랜덤 블링크 (svg 스쿼시 0.88)

### 3. 움직임 범위 확대
- 기존: ±12~18px 양옆 미세 움직임
- 개선: ±50~100px 전방위 떠다님
- 애니메이션 5종 (f1~f5), 각 캐릭터마다 속도/딜레이 다름
- 회전도 섞임 (±3~5deg)
- 브레인은 32초 주기 자유 배회

### 4. 호버 툴팁 복원
- 얼굴 위쪽에 말풍선 형태 팝업 (화살표 아래)
- 어두운 배경 + accent 컬러 테두리
- 15자 이내 짧은 후크
- 데스크톱: hover, 모바일: 첫 탭 = 툴팁, 두 번째 탭 = 모달

### 5. 콘텐츠 선별 (재미 기준)
당연한 내용·시시한 디테일(예: "19자 건조") 전부 배제. 흥미 유발 기준:
- **비판을 무서워 않는 것만 우월** (교수 대면 일화)
- **모의고사 1등급 / 수능 3등급** (극단적 상태 의존성)
- **애인 무릎을 직접 일으켰다 → 3일 뒤 이별** (권력 역전의 물리적 증거)
- **버스가 안 올 걸 알면서 기다렸다** (선제 비관의 생존 전략)
- **게임 상위 0.1%, 본캐만** (확신 유무의 이분법)
- **식빵 나눔은 의무에 가까운 의식** (자기 행동을 실시간 진단)
- 유지: 지옥을 같이 갈 수 있는 것, 분석은 헌신, 논리 필터링, 성공/실패 구분, AI 코드공부 시간낭비, 아부 싫어함, 불편하고 싶다, 가시와 사랑, 노무사, 데이터에서 사람 읽는다, 데이터 주제

**애인 이름 비공개** — "정원이"는 전부 "애인"으로 추상화.

### 6. 해 모드 = 온라인 이력서
- Projects (창작) / **Tools & Utilities (해야 해서 만든 것)** / How I Work (워크플로우) / Skills / Philosophy at Work / Interests / Contact
- **수특 문제 생성기·운동 PWA는 Tools 섹션으로 분리** (창작물과 혼동 제거)
- Philosophy는 업무 관련 3개만 (코드 공부 안 함 / 한 AI 믿지 않음 / 판례 불수정)
- 스크롤 시 카드 페이드인 (IntersectionObserver)
- 프로젝트 카드에 기술 스택 메타 표시

### 7. 기타 디테일
- 파티클 수 180 → 200, 색상 hue 변주 (160~220)
- 패럴랙스 강도 8px → 10px
- 모달 쿼트 텍스트에 accent 컬러 따옴표 장식
- 모달 태그 라벨 추가 (주제/태도/철학/관계관/사건/자기이해/에피소드)
- 모달 blur 8px → 10px, 스케일 애니메이션 cubic-bezier
- 파비콘 교체 (뇌 → 눈이 큰 둥근 얼굴)
